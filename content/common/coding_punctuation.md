# Coding Punctuation

| Symbol                            | Name(s)          |
| --------------------------------- | ---------------- |
| **Enclosing Punctuation**         |
| `(` `)`                           | Parentheses      |
| `{` `}`                           | Curly Braces     |
| `[` `]`                           | Square Brackets  |
| `<` `>`                           | Angle Brackets   |
| `"`                               | Double Quotes    |
| `'`                               | Single Quotes    |
| <code>&#96;</code>                | Backtick         |
| **Operators and Logical Symbols** |
| `=`                               | Equals Sign      |
| `+`                               | Plus Sign        |
| `-`                               | Dash, Hyphen     |
| `*`                               | Asterisk         |
| `/`                               | Forward Slash    |
| `\`                               | Backslash        |
| `%`                               | Percent, Modulus |
| `^`                               | Caret            |
| `&`                               | Ampersand        |
