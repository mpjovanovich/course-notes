# CRUD

**CRUD** describes the four main Data Manipulation Language (DML) operations:

| Operation | SQL Command |
| --------- | ----------- |
| Create    | INSERT      |
| Read      | SELECT      |
| Update    | UPDATE      |
| Delete    | DELETE      |

For syntax reference:

- [SQL Insert](https://www.sqlitetutorial.net/sqlite-insert/)
- [SQL Update](https://www.sqlitetutorial.net/sqlite-update/)
- [SQL Delete](https://www.sqlitetutorial.net/sqlite-delete/)
